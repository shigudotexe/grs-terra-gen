## Analysis

Biomes without crops are primarily:
1. **Aquatic biomes** (15) - Oceans, rivers, deep ocean. These naturally don't support crops
2. **Cave biomes** (3) - Underground environments naturally don't support traditional crops
3. **Coastal biomes** - Beach and sea-related biomes typically don't have crop features
4. **Specialized biomes** - Volcanic and mushroom biomes don't have crop features (thematic design)
5. **Terrain variants** - Some flat/hill/mountain variants that extend from base biomes without crop inheritance
